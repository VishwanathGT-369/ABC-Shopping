ABC EShopping application developed by Ms. Vinuta Soratoor(vinutavvsoratoor@gmail.com)

> This is a command line application. You can run the program from "com/abc/EShoppingApplicationDemo.java"
> It uses MySql as database. File "sql_queries.txt" contains all DDL and DML queries to create db, tables and initial data to bootstrap the application.
> You can find db configurations in "resources/config.properties" file.
  You can change database name, port, user name and password according to your setup
